📖 دفتر روزنامه چیست؟

هر رویداد مالی، نخستین بار در دفتر روزنامه (General Journal) ثبت می‌شود. این کار ثبت اولیه نام دارد. ثبت‌ها به ترتیب تاریخ و بر اساس اسناد مثبته (فاکتور، رسید، چک) انجام می‌شوند.

📋 ستون‌های اصلی

  1. 📅 تاریخ
  2. 📝 شرح (نام حساب‌ها)
  3. 🔢 شماره سند
  4. ⬅️ بدهکار (چپ)
  5. ➡️ بستانکار (راست)

⚖️ قوانین طلایی

  1. همواره بدهکار = بستانکار
  2. ابتدا حساب بدهکار ← سپس بستانکار
  3. حداقل دو حساب درگیر می‌شوند
  4. توضیح کوتاه زیر ثبت نوشته شود

🧠 آزمونک جلسه دوم

📌 جمع‌بندی جلسه

  1. دفتر روزنامه = محل ثبت اولیه رویدادهای مالی.
  2. هر رویداد یک بدهکار و یک بستانکار دارد.
  3. بدهکار (چپ) ← دارایی‌ها و هزینه‌ها.
  4. بستانکار (راست) ← بدهی‌ها، سرمایه و درآمدها.
  5. همواره جمع بدهکار = جمع بستانکار.
